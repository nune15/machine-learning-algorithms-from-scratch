# Machine Learning Algorithms From Scratch (Python)

This repository contains implementations of core ML algorithms **from scratch** in Python.  
All algorithms are implemented manually — no external ML libraries used.

## 🔹 Algorithms Included

- **Decision Tree Classifier**  
  Splitting using Gini Index. Handles categorical and numerical data.
- **K-Nearest Neighbors (KNN)**  
  Classification using Euclidean distance. K is configurable.
- **Gini Index Calculation**  
  Core function used for decision tree splitting.
